# 網頁作品集

盡量用純 HTML、CSS、Javascript 做出現階段自己能做出的效果

## 目錄

-   **Game** : 遊戲
    -   **Go** : 圍棋
    -   **Minesweeper** : 踩地雷
-   **Canvas** : Canvas
    -   **Fireworks** : 煙火特效
-   **Scroll** : 滑鼠滾動特效
    -   **Animation** : 動畫版
    -   **Mandatory** : 投影片版
-   **Menu** : 菜單動畫
    -   **Surround** : 環繞版
-   **WebDemo** : 網頁展示
